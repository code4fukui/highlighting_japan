# highlighting_japan

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

An open dataset (CSV/JSON) of the article index from *Highlighting Japan*, the Japanese government's publicity magazine. This repository provides machine-readable data for articles published between 2011 and 2022, including auto-generated keywords from the Japanese article text.

## Sample App

An interactive viewer and search interface for the dataset:

- **https://github.com/code4fukui/highlighting_japan

## Open Data

The dataset is generated by scraping the official *Highlighting Japan* website and is available in two formats.

- [data_ja.csv](https://code4fukui.github.io/highlighting_japan/data_ja.csv)
- [data_ja.json](https://code4fukui.github.io/highlighting_japan/data_ja.json)

Each record contains the article's URL, title, publication date, description, image URL, and a list of keywords extracted using [TinySegmenter](https://code4fukui.github.io/TinySegmenter/TinySegmenter.js).

## Development

The data is generated by the Deno script in `deno/download.js`. To run it and regenerate the dataset:

```sh
deno run --allow-net --allow-read --allow-write deno/download.js
```
